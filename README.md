# Satzbauer



## What can you do with it?

- Become a German grammar freak
- Learn the difference between verb conjugations, adjectives/adverbs, nouns, pronouns ...
- Works with any German text in docx, txt, pdf, epub, pptx format
- No coding required!
- Download,install and start
- Source code included!
- For Windows!
